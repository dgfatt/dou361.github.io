# dou361.github.io
个人博客

blog结构说明


	_data			网站关键字语言国际化
	_drafts			草稿临时文件
	_includes
	_layouts		布局
	_plugins		插件
	_posts			帖子markdown文章存放
	_site			jekyll方式生成的网站内容
	about
	archives		归档
	assets			网站基本的网页内容
	categories		分类
	search			搜索
	tags			标签
	_config.xml		网站配置文件
	404.html		外链的腾讯公益404页面
	atom.xml		订阅网志的格式和rss类似
	CNAME			别名域名跳转使用
	Gemfile			gem文件
	Gemfile.lock	gem文件上锁，如果需要重新安装需要删除该文件
	index.html		首页
	rss.xml			订阅
	sitemap.txt		站点地图
